Privacy Policy

Last updated: January 2025

TL;DR: X Auto Sort Extension does not collect, store, or transmit any personal data. Everything happens locally in your browser.

What We Don't Collect

X Auto Sort Extension is designed with privacy in mind. We do not collect:

  • Personal information
  • Browsing history
  • X.com (Twitter) account data
  • Tweets or messages you view
  • IP addresses
  • Analytics or usage data

What the Extension Does

The extension only:

  • Modifies X.com page requests locally in your browser to sort replies by likes
  • Stores your preference (enabled/disabled) locally in your browser
  • Counts sorted tweets locally for display in the popup (never transmitted)

Data Storage

All data is stored locally using Chrome's storage API:

  • Enabled state: Whether the extension is on or off
  • Sorted count: Number of tweets sorted (for display only)

This data never leaves your device and can be cleared by uninstalling the extension.

Permissions Used

  • Host permissions (x.com, twitter.com): Required to modify page behavior on X.com
  • Storage: To save your preferences locally
  • Scripting: To inject the sorting functionality

Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

Open Source

This extension is open source. You can review the entire source code on GitHub to verify our privacy practices.

Changes to This Policy

Any changes to this privacy policy will be reflected in the extension update notes and on this page.

Contact

If you have any questions about this privacy policy, please open an issue on our GitHub repository.

Your privacy is important to us. This extension was built to be useful without compromising your data.
